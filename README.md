# Wounded

**Wounded** — десктопное приложение, аналог Apple Music, построенное на Electron и React. Проект выполнен в рамках учебной работы.

**Автор:** Литвин Владислав, студент НИУ МИЭТ, группа ПИН-33

## Технологии

- React, Redux Toolkit, TypeScript
- Electron
- Webpack
- Jest (тестирование)
- ESLint, Prettier (линтинг и форматирование)

## Запуск в режиме разработки

Для разработки с горячей перезагрузкой выполните следующие шаги:

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Выполните сборку клиентской части в режиме разработки:
   ```bash
   npm run build:dev
   ```

3. Запустите Webpack Dev Server:
   ```bash
   npm run start
   ```

4. В отдельном терминале запустите Electron:
   ```bash
   npm run electron
   ```

После этого откроется окно приложения. Изменения в коде будут автоматически применяться.

## Получение готового приложения

Будет позже

### Сборка вручную

Если вы хотите собрать приложение самостоятельно:

1. Установите зависимости (если не сделали ранее):
   ```bash
   npm install
   ```

2. Соберите production-версию клиентской части:
   ```bash
   npm run build:prod
   ```

3. Транспилируйте код Electron:
   ```bash
   npm run transpile:electron
   ```

4. Запустите сборку под нужную платформу:
   - **macOS (ARM64):** `npm run dist:mac`
   - **Linux (x64):** `npm run dist:linux`
   - **Windows (x64):** `npm run dist:win`

Готовые дистрибутивы появятся в папке `dist`.

## Дополнительные команды

- **Запуск тестов:** `npm test`
- **Проверка линтером:** `npm run lint`
- **Автоматическое исправление ошибок и форматирование:** `npm run lint:fix`
- **Форматирование кода:** `npm run format`
- **Проверка форматирования:** `npm run format:check`

## Примечания

- При разработке убедитесь, что порт 3000 (используемый Webpack Dev Server) свободен. Либо при запуске укажите свой порт как переменную окружения. Пример:
  ```bash
  npm run start -- --port=3001
  ```
- При создании Pull Request в репозитории автоматически запускаются линтеры, тесты и проверка сборки (GitHub Actions).
